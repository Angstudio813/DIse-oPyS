🚀 Cómo Ejecutar el Proyecto

  1. Navegar a la carpeta del proyecto: cd DIseñoPyS
  2. Instalar dependencias: npm install
  3. Ejecutar el servidor: npm start
  4. Abrir el navegador en: http://localhost:3000
